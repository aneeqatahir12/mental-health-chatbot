# Contributing to Mental Health Chatbot

Thank you for wanting to improve this project!

---

## 🐛 Found a Bug?
1. Click the **"Issues"** tab on GitHub
2. Click **"New Issue"**
3. Describe what went wrong and include a screenshot if possible

## 💡 Want to Add Something?
Great ideas:
- Add more WHO documents
- Support languages other than English
- Add a web chat interface
- Connect to WhatsApp or Telegram

To contribute:
1. Fork this repo
2. Make your changes
3. Submit a Pull Request with a clear description

## 📋 Guidelines
- Keep node names simple and clear
- Test your workflow before submitting
- Add comments to explain anything complex
